# Google Test (GTest)

- Basic introduction to unit testing with Google Test (GTest)

---

## How to build

Requirements: Google Test (GTest)

```bash
mkdir build
cd build
cmake ..
make
```

```docker
docker build -t slam:8:2
docker run -it slam:8_2

# inside docker
cd /fastcampus_slam_codes/8_2/build
./unitTests
```
